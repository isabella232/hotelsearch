# hotelsearch

Api
======
Using node 6.1.0, install the dependencies:
- cd server
- npm install

Running
---------
npm start

Vagrant
----
The api can also be executed with vagrant + ansible provisioner by running
- vagrant up
- The api will be available at http://192.168.50.3:3030/hotels

Front
======

Using Ruby 2.3.0:

- bundle install
- rails s
